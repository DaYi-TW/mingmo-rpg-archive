## 🟥 清廷兵力動員評估

> 判定：清廷三藩之亂後常備軍約70萬～80萬級，其中綠營約60萬上下、八旗約12萬～20萬。清廷可調戰爭體量40萬+，但受糧草、道路、渡河、馬料、營地與地方駐防限制，不可能把40萬人一次性塞到武昌北線壕溝前。聯邦需按「全國常備70萬級、可調戰爭體量40萬+、湖北主戰場10萬級」的尺度準備。

| 層級 | 兵力規模 | 說明 |
|------|------:|------|
| 清廷全國常備軍 | 70萬～80萬 | 八旗約12萬～20萬、綠營約60萬上下；含全國駐防，不等於可集中南下 |
| 可調戰爭體量 | 40萬+ | 三藩已平後能調出大量八旗/綠營/地方兵力與民夫，但需分批、分省、分戰場使用 |
| 北線前鋒試探 | 2,000～5,000 | 已見約2,000前鋒體系；十一月初七夜襲與十一月初八山道探路證明其用途為測夜防、測砲位、抓火力反應 |
| 秋季試探/壓迫兵團 | 1萬～3萬 | 前線可見火營、糧車、馬圈與夜襲體量支持此判斷；不是40萬大軍壓境 |
| 湖北方向正式戰役主力 | 8萬～15萬 | 若糧台、渡河器材、馬料補齊，可對武昌—湖北北線發動正式戰役 |
| 湖北方向極限壓迫 | 15萬～20萬 | 康熙若高度重視並長期準備，可能提高湖北方向投入，但補給壓力極大 |
| 多省牽制與側翼兵力 | 5萬～10萬 | 可能牽制湖南、江西、安徽、福建等方向，分散聯邦機動兵力 |
| 後勤、糧台、運輸與輪換 | 5萬～10萬 | 民夫、馬夫、輜重兵、糧站守備、道路橋渡修築 |
| 號稱/累計動員兵力 | 40萬+ | 可作為戰爭宣稱、累計調發或多戰場總數，不等於單一戰場可投送兵力 |

### 戰略含義

- 清廷「常備70萬～80萬」與「能在北線同時投入70萬」不是同一件事。
- 清廷「能調40萬+戰爭體量」與「能在北線同時投入40萬」也不是同一件事。
- 聯邦最危險的不是單場技術對決，而是清廷以10萬級主力反覆壓迫、以40萬+可調體量支撐輪換與消耗。
- 北線若守住數次試探，清廷可能改從側翼、糧道、滲透、謠言與多省牽制下手。
- 聯邦應避免深追，利用鐵路、電報、飛艇、ZPC-1與JQ-3在防線內打高效率殲耗。

### 清軍武器結構補充

- 清軍不是純冷兵器軍隊，但主體仍是冷兵器、弓箭、騎兵、盾車、藤牌與民夫工程；鳥銃、抬槍、舊式火炮、火器營存在，但不是全軍主導武器。
- JQ-3後膛槍有效射程400m、8發/分，對清軍鳥銃具3-4倍有效射程優勢與壓倒性射速優勢。
- 清軍鳥銃若在百米外散射，可造成流彈、跳彈、心理壓力與偶發傷亡，但不能穩定壓制聯邦壕線。
- 清廷合理戰法是用霧、夜間、盾車、浮橋、民夫填壕、騎兵側壓與多點同壓縮短距離；若正面在300-400m暴露隊形，會被聯邦步槍與砲兵高效率殺傷。

### 戰略含義

- 聯邦不宜主動深追，應利用壕溝、防線、砲兵、飛艇、後膛槍和電報打防禦殲耗戰。
- 清廷若正面衝擊北線，將承受極高火力與通訊劣勢。
- 清廷威脅不在單場戰術，而在人口、糧草、縱深、地方滲透與長期消耗。
- 聯邦最佳打法：空偵預警、假情報誘敵、砲兵殺傷、後膛槍火力牆、騎兵斷糧、艦砲封鎖、鐵路/電報快速調度。
